–ù–∏–∫–æ–≥–¥–∞ –Ω–µ –¥–µ–ª–∞–π—Ç–µ `rebase` –≤–µ—Ç–æ–∫, –∫–æ—Ç–æ—Ä—ã–µ –∫—Ç–æ-—Ç–æ –∫—Ä–æ–º–µ –≤–∞—Å –∏—Å–ø–æ–ª—å–∑—É–µ—Ç (–Ω–∞–ø—Ä–∏–º–µ—Ä, –æ–±—â—É—é –≤–µ—Ç–∫—É `main` –∏–ª–∏ –≤–µ—Ç–∫—É –∫–æ–ª–ª–µ–≥–∏). –≠—Ç–æ –º–µ–Ω—è–µ—Ç –∏—Å—Ç–æ—Ä–∏—é –∏ —Å–æ–∑–¥–∞—Å—Ç —Ö–∞–æ—Å —É –¥—Ä—É–≥–∏—Ö —Ä–∞–∑—Ä–∞–±–æ—Ç—á–∏–∫–æ–≤.

---

## ‚ùì –í–æ–ø—Ä–æ—Å—ã —Å —Å–æ–±–µ—Å–µ–¥–æ–≤–∞–Ω–∏–π (–¢–û–ü-10)

–≠—Ç–æ—Ç —Ä–∞–∑–¥–µ–ª –ø–æ–º–æ–∂–µ—Ç –≤–∞–º –ø–æ–¥–≥–æ—Ç–æ–≤–∏—Ç—å—Å—è –∫ —Ç–µ—Ö–Ω–∏—á–µ—Å–∫–æ–º—É –∏–Ω—Ç–µ—Ä–≤—å—é. –í–æ–ø—Ä–æ—Å—ã –æ—Ö–≤–∞—Ç—ã–≤–∞—é—Ç –∫–∞–∫ –±–∞–∑–æ–≤—ã–µ –∫–æ–Ω—Ü–µ–ø—Ü–∏–∏, —Ç–∞–∫ –∏ —Ä–µ–∞–ª—å–Ω—ã–µ —Ä–∞–±–æ—á–∏–µ —Å–∏—Ç—É–∞—Ü–∏–∏.

### 1. –í —á–µ–º —Ä–∞–∑–Ω–∏—Ü–∞ –º–µ–∂–¥—É `git merge` –∏ `git rebase`?
**–û—Ç–≤–µ—Ç:**
*   **`git merge`** –æ–±—ä–µ–¥–∏–Ω—è–µ—Ç –¥–≤–µ –≤–µ—Ç–∫–∏, —Å–æ–∑–¥–∞–≤–∞—è –Ω–æ–≤—ã–π "–∫–æ–º–º–∏—Ç —Å–ª–∏—è–Ω–∏—è". –û–Ω —Å–æ—Ö—Ä–∞–Ω—è–µ—Ç –ø–æ–ª–Ω—É—é –∏—Å—Ç–æ—Ä–∏—é –∏ –ø–æ–∫–∞–∑—ã–≤–∞–µ—Ç, –∫–æ–≥–¥–∞ –∏ –∫–∞–∫–∏–µ –≤–µ—Ç–∫–∏ —Å–ª–∏–≤–∞–ª–∏—Å—å. –≠—Ç–æ –±–µ–∑–æ–ø–∞—Å–Ω–µ–µ –¥–ª—è –∏—Å—Ç–æ—Ä–∏–∏, –∫–æ—Ç–æ—Ä–∞—è —É–∂–µ –µ—Å—Ç—å –Ω–∞ —Å–µ—Ä–≤–µ—Ä–µ.
*   **`git rebase`** –ø–µ—Ä–µ–ø–∏—Å—ã–≤–∞–µ—Ç –∏—Å—Ç–æ—Ä–∏—é —Ç–µ–∫—É—â–µ–π –≤–µ—Ç–∫–∏, –ø–µ—Ä–µ–Ω–æ—Å—è –µ–µ –∫–æ–º–º–∏—Ç—ã –≤ –∫–æ–Ω–µ—Ü —Ü–µ–ª–µ–≤–æ–π –≤–µ—Ç–∫–∏. –≠—Ç–æ —Å–æ–∑–¥–∞–µ—Ç –ª–∏–Ω–µ–π–Ω—É—é, —á–∏—Å—Ç—É—é –∏—Å—Ç–æ—Ä–∏—é –±–µ–∑ –ª–∏—à–Ω–∏—Ö –∫–æ–º–º–∏—Ç–æ–≤ —Å–ª–∏—è–Ω–∏—è, –Ω–æ –Ω–µ —Å—Ç–æ–∏—Ç –ø—Ä–∏–º–µ–Ω—è—Ç—å –∫ –≤–µ—Ç–∫–∞–º, —Å –∫–æ—Ç–æ—Ä—ã–º–∏ —Ä–∞–±–æ—Ç–∞—é—Ç –¥—Ä—É–≥–∏–µ –ª—é–¥–∏.

### 2. –ö–∞–∫ —Ä–∞–∑—Ä–µ—à–∏—Ç—å –∫–æ–Ω—Ñ–ª–∏–∫—Ç –≤ Git?
**–û—Ç–≤–µ—Ç:**
1.  Git –ø–æ–º–µ—Ç–∏—Ç –∫–æ–Ω—Ñ–ª–∏–∫—Ç—É—é—â–∏–µ —Ñ–∞–π–ª—ã. –í—ã —É–≤–∏–¥–∏—Ç–µ —ç—Ç–æ –≤ —Å—Ç–∞—Ç—É—Å–µ (`git status`) ‚Äî `both modified`.
2.  –û—Ç–∫—Ä—ã—Ç—å –∫–æ–Ω—Ñ–ª–∏–∫—Ç—É—é—â–∏–π —Ñ–∞–π–ª. Git –æ—Ç–º–µ—Ç–∏—Ç —É—á–∞—Å—Ç–∫–∏ –∫–æ–Ω—Ñ–ª–∏–∫—Ç–∞ —Ç–∞–∫:
    ```
    <<<<<<< HEAD
    –≤–∞—à –∫–æ–¥ –∏–∑ —Ç–µ–∫—É—â–µ–π –≤–µ—Ç–∫–∏
    =======
    –∫–æ–¥ –∏–∑ –≤–ª–∏–≤–∞–µ–º–æ–π –≤–µ—Ç–∫–∏
    >>>>>>> feature-branch
    ```
3.  –í—Ä—É—á–Ω—É—é –æ—Ç—Ä–µ–¥–∞–∫—Ç–∏—Ä–æ–≤–∞—Ç—å —Ñ–∞–π–ª, –æ—Å—Ç–∞–≤–∏–≤ –Ω—É–∂–Ω—ã–π –∫–æ–¥ –∏ —É–¥–∞–ª–∏–≤ –º–∞—Ä–∫–µ—Ä—ã (`<<<`, `===`, `>>>`).
4.  –°–æ—Ö—Ä–∞–Ω–∏—Ç—å —Ñ–∞–π–ª, –¥–æ–±–∞–≤–∏—Ç—å –µ–≥–æ –≤ –∏–Ω–¥–µ–∫—Å (`git add <—Ñ–∞–π–ª>`).
5.  –ü—Ä–æ–¥–æ–ª–∂–∏—Ç—å –æ–ø–µ—Ä–∞—Ü–∏—é: –µ—Å–ª–∏ –±—ã–ª `merge` ‚Äî —Å–¥–µ–ª–∞—Ç—å `git commit`, –µ—Å–ª–∏ `rebase` ‚Äî `git rebase --continue`.

### 3. –ß–µ–º –æ—Ç–ª–∏—á–∞–µ—Ç—Å—è `git pull` –æ—Ç `git fetch`?
**–û—Ç–≤–µ—Ç:**
*   **`git fetch`** –∑–∞–≥—Ä—É–∂–∞–µ—Ç –Ω–æ–≤—ã–µ –¥–∞–Ω–Ω—ã–µ –∏–∑ —É–¥–∞–ª–µ–Ω–Ω–æ–≥–æ —Ä–µ–ø–æ–∑–∏—Ç–æ—Ä–∏—è, –Ω–æ **–Ω–µ —Å–ª–∏–≤–∞–µ—Ç** –∏—Ö —Å –≤–∞—à–µ–π –ª–æ–∫–∞–ª—å–Ω–æ–π –≤–µ—Ç–∫–æ–π. –í–∞—à–∞ —Ä–∞–±–æ—á–∞—è –∫–æ–ø–∏—è –æ—Å—Ç–∞–µ—Ç—Å—è –Ω–µ—Ç—Ä–æ–Ω—É—Ç–æ–π. –≠—Ç–æ –±–µ–∑–æ–ø–∞—Å–Ω—ã–π —Å–ø–æ—Å–æ–± –ø–æ—Å–º–æ—Ç—Ä–µ—Ç—å, —á—Ç–æ –Ω–æ–≤–æ–≥–æ —É –∫–æ–ª–ª–µ–≥.
*   **`git pull`** –¥–µ–ª–∞–µ—Ç —Å—Ä–∞–∑—É –¥–≤–µ –≤–µ—â–∏: —Å–Ω–∞—á–∞–ª–∞ `git fetch` (–∑–∞–≥—Ä—É–∂–∞–µ—Ç –¥–∞–Ω–Ω—ã–µ), –∞ –∑–∞—Ç–µ–º `git merge` (–ø—ã—Ç–∞–µ—Ç—Å—è –≤–ª–∏—Ç—å —ç—Ç–∏ –∏–∑–º–µ–Ω–µ–Ω–∏—è –≤ –≤–∞—à—É —Ç–µ–∫—É—â—É—é –≤–µ—Ç–∫—É).

### 4. –ß—Ç–æ —Ç–∞–∫–æ–µ `.gitignore` –∏ –∑–∞—á–µ–º –æ–Ω –Ω—É–∂–µ–Ω?
**–û—Ç–≤–µ—Ç:**
–≠—Ç–æ —Å–ø–µ—Ü–∏–∞–ª—å–Ω—ã–π —Ñ–∞–π–ª, –≤ –∫–æ—Ç–æ—Ä–æ–º –ø–µ—Ä–µ—á–∏—Å–ª—è—é—Ç—Å—è —à–∞–±–ª–æ–Ω—ã —Ñ–∞–π–ª–æ–≤ –∏ –ø–∞–ø–æ–∫, –∫–æ—Ç–æ—Ä—ã–µ Git –¥–æ–ª–∂–µ–Ω –∏–≥–Ω–æ—Ä–∏—Ä–æ–≤–∞—Ç—å. –≠—Ç–æ –Ω—É–∂–Ω–æ, —á—Ç–æ–±—ã –Ω–µ –∑–∞—Å–æ—Ä—è—Ç—å —Ä–µ–ø–æ–∑–∏—Ç–æ—Ä–∏–π —Å–ª—É–∂–µ–±–Ω—ã–º–∏ —Ñ–∞–π–ª–∞–º–∏ (–ø–∞–ø–∫–∏ IDE `.idea/`), –≤—Ä–µ–º–µ–Ω–Ω—ã–º–∏ —Ñ–∞–π–ª–∞–º–∏, –∫–µ—à–µ–º, –≤–∏—Ä—Ç—É–∞–ª—å–Ω—ã–º–∏ –æ–∫—Ä—É–∂–µ–Ω–∏—è–º–∏ (`venv/`, `node_modules/`), —Ñ–∞–π–ª–∞–º–∏ —Å –ø–∞—Ä–æ–ª—è–º–∏ –∏ —Ç.–¥.

### 5. –¢—ã —Å–ª—É—á–∞–π–Ω–æ —Å–¥–µ–ª–∞–ª `commit` –Ω–µ –≤ —Ç—É –≤–µ—Ç–∫—É. –ö–∞–∫ —ç—Ç–æ –∏—Å–ø—Ä–∞–≤–∏—Ç—å?
**–û—Ç–≤–µ—Ç:**
–°–∏—Ç—É–∞—Ü–∏—è: –Ω—É–∂–Ω–æ –±—ã–ª–æ –∫–æ–º–º–∏—Ç–∏—Ç—å –≤ `feature`, –∞ –∑–∞–∫–æ–º–º–∏—Ç–∏–ª–∏ –≤ `main`.
1.  –ü–µ—Ä–µ–∫–ª—é—á–∏—Ç—å—Å—è –Ω–∞ –ø—Ä–∞–≤–∏–ª—å–Ω—É—é –≤–µ—Ç–∫—É: `git checkout feature`
2.  –ó–∞–±—Ä–∞—Ç—å –ø–æ—Å–ª–µ–¥–Ω–∏–π –∫–æ–º–º–∏—Ç –∏–∑ `main`: `git cherry-pick main` (–∏–ª–∏ –∏—Å–ø–æ–ª—å–∑—É—è —Ö–µ—à –∫–æ–º–º–∏—Ç–∞).
3.  –í–µ—Ä–Ω—É—Ç—å—Å—è –Ω–∞ –≤–µ—Ç–∫—É `main` –∏ –æ—Ç–∫–∞—Ç–∏—Ç—å –µ–µ –Ω–∞ –∫–æ–º–º–∏—Ç –Ω–∞–∑–∞–¥, —É–¥–∞–ª–∏–≤ "–Ω–µ–Ω—É–∂–Ω—ã–π" –∫–æ–º–º–∏—Ç: `git checkout main` –∏ `git reset --hard HEAD~1`.

### 6. –ß—Ç–æ —Ç–∞–∫–æ–µ `git stash` –∏ –∫–æ–≥–¥–∞ –µ–≥–æ –ø—Ä–∏–º–µ–Ω—è—é—Ç?
**–û—Ç–≤–µ—Ç:**
`git stash` –ø–æ–∑–≤–æ–ª—è–µ—Ç –≤—Ä–µ–º–µ–Ω–Ω–æ "–æ—Ç–ª–æ–∂–∏—Ç—å" –Ω–µ–∑–∞–∫–æ–º–º–∏—á–µ–Ω–Ω—ã–µ –∏–∑–º–µ–Ω–µ–Ω–∏—è –≤ —Å—Ç–æ—Ä–æ–Ω—É –∏ –≤–µ—Ä–Ω—É—Ç—å —Ä–∞–±–æ—á—É—é –¥–∏—Ä–µ–∫—Ç–æ—Ä–∏—é –∫ —á–∏—Å—Ç–æ–º—É —Å–æ—Å—Ç–æ—è–Ω–∏—é (–∫–∞–∫ –≤ –ø–æ—Å–ª–µ–¥–Ω–µ–º –∫–æ–º–º–∏—Ç–µ). –≠—Ç–æ –ø–æ–ª–µ–∑–Ω–æ, –∫–æ–≥–¥–∞ –≤–∞–º –Ω—É–∂–Ω–æ —Å—Ä–æ—á–Ω–æ –ø–µ—Ä–µ–∫–ª—é—á–∏—Ç—å—Å—è –Ω–∞ –¥—Ä—É–≥—É—é –≤–µ—Ç–∫—É –∏–ª–∏ –∑–∞–¥–∞—á—É, –Ω–æ –≤—ã –Ω–µ —Ö–æ—Ç–∏—Ç–µ –∫–æ–º–º–∏—Ç–∏—Ç—å –Ω–µ–¥–æ–¥–µ–ª–∞–Ω–Ω—É—é —Ä–∞–±–æ—Ç—É. –ü–æ—Ç–æ–º –º–æ–∂–Ω–æ –≤–µ—Ä–Ω—É—Ç—å –∏–∑–º–µ–Ω–µ–Ω–∏—è –∫–æ–º–∞–Ω–¥–æ–π `git stash pop`.

### 7. –ö–∞–∫ –∏—Å–ø—Ä–∞–≤–∏—Ç—å —Å–æ–æ–±—â–µ–Ω–∏–µ –ø–æ—Å–ª–µ–¥–Ω–µ–≥–æ –∫–æ–º–º–∏—Ç–∞, –µ—Å–ª–∏ –≤—ã –æ—à–∏–±–ª–∏—Å—å?
**–û—Ç–≤–µ—Ç:**
–ù—É–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –∫–æ–º–∞–Ω–¥—É `git commit --amend -m "–ù–æ–≤–æ–µ –ø—Ä–∞–≤–∏–ª—å–Ω–æ–µ —Å–æ–æ–±—â–µ–Ω–∏–µ"`. –≠—Ç–∞ –∫–æ–º–∞–Ω–¥–∞ –∑–∞–º–µ–Ω–∏—Ç –ø–æ—Å–ª–µ–¥–Ω–∏–π –∫–æ–º–º–∏—Ç –Ω–æ–≤—ã–º (—Å –Ω–æ–≤—ã–º —Ö–µ—à–µ–º). –ï—Å–ª–∏ –∫–æ–º–º–∏—Ç —É–∂–µ –±—ã–ª –∑–∞–ø—É—à–µ–Ω –Ω–∞ —Å–µ—Ä–≤–µ—Ä, –ø–æ—Å–ª–µ `amend` –ø–æ—Ç—Ä–µ–±—É–µ—Ç—Å—è –¥–µ–ª–∞—Ç—å `git push --force-with-lease` (—Å –æ—Å—Ç–æ—Ä–æ–∂–Ω–æ—Å—Ç—å—é!).

### 8. –û–±—ä—è—Å–Ω–∏, —á—Ç–æ —Ç–∞–∫–æ–µ `HEAD` –≤ Git.
**–û—Ç–≤–µ—Ç:**
`HEAD` ‚Äî —ç—Ç–æ —É–∫–∞–∑–∞—Ç–µ–ª—å (—Å—Å—ã–ª–∫–∞) –Ω–∞ —Ç–µ–∫—É—â–∏–π –∫–æ–º–º–∏—Ç –∏–ª–∏ –≤–µ—Ç–∫—É, –≤ –∫–æ—Ç–æ—Ä–æ–π –≤—ã —Å–µ–π—á–∞—Å –Ω–∞—Ö–æ–¥–∏—Ç–µ—Å—å. –û–±—ã—á–Ω–æ –æ–Ω —É–∫–∞–∑—ã–≤–∞–µ—Ç –Ω–∞ –∏–º—è –≤–µ—Ç–∫–∏ (–Ω–∞–ø—Ä–∏–º–µ—Ä, `refs/heads/main`), –∞ –≤–µ—Ç–∫–∞ —É–∂–µ —É–∫–∞–∑—ã–≤–∞–µ—Ç –Ω–∞ –∫–æ–Ω–∫—Ä–µ—Ç–Ω—ã–π –∫–æ–º–º–∏—Ç. –ï—Å–ª–∏ –ø–µ—Ä–µ–∫–ª—é—á–∏—Ç—å—Å—è –Ω–∞ –∫–æ–Ω–∫—Ä–µ—Ç–Ω—ã–π –∫–æ–º–º–∏—Ç (–∞ –Ω–µ –≤–µ—Ç–∫—É), Git —Å–æ–æ–±—â–∏—Ç, —á—Ç–æ –≤—ã –≤ —Å–æ—Å—Ç–æ—è–Ω–∏–∏ "detached HEAD".

### 9. –ö–∞–∫ —É–¥–∞–ª–∏—Ç—å —Ñ–∞–π–ª –∏–∑ —Ä–µ–ø–æ–∑–∏—Ç–æ—Ä–∏—è, –Ω–æ –æ—Å—Ç–∞–≤–∏—Ç—å –µ–≥–æ –Ω–∞ –¥–∏—Å–∫–µ?
**–û—Ç–≤–µ—Ç:**
–û–±—ã—á–Ω–æ `git rm <—Ñ–∞–π–ª>` —É–¥–∞–ª—è–µ—Ç —Ñ–∞–π–ª –∏ –∏–∑ Git, –∏ —Å –¥–∏—Å–∫–∞. –ß—Ç–æ–±—ã —É–±—Ä–∞—Ç—å —Ñ–∞–π–ª —Ç–æ–ª—å–∫–æ –∏–∑-–ø–æ–¥ –≤–µ—Ä—Å–∏–æ–Ω–Ω–æ–≥–æ –∫–æ–Ω—Ç—Ä–æ–ª—è, –Ω–æ –æ—Å—Ç–∞–≤–∏—Ç—å –µ–≥–æ –≤ –ø–∞–ø–∫–µ –ø—Ä–æ–µ–∫—Ç–∞ (–Ω–∞–ø—Ä–∏–º–µ—Ä, –¥–ª—è –ª–æ–∫–∞–ª—å–Ω—ã—Ö –Ω–∞—Å—Ç—Ä–æ–µ–∫), –Ω—É–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å —Ñ–ª–∞–≥ `--cached`:
`git rm --cached <—Ñ–∞–π–ª>`
–ü–æ—Å–ª–µ —ç—Ç–æ–≥–æ –Ω—É–∂–Ω–æ –¥–æ–±–∞–≤–∏—Ç—å —ç—Ç–æ—Ç —Ñ–∞–π–ª –≤ `.gitignore`, —á—Ç–æ–±—ã Git —Å–Ω–æ–≤–∞ –Ω–µ –Ω–∞—á–∞–ª –µ–≥–æ –æ—Ç—Å–ª–µ–∂–∏–≤–∞—Ç—å.

### 10. –ö–∞–∫–∏–µ —Å—Ç—Ä–∞—Ç–µ–≥–∏–∏ –≤–µ—Ç–≤–ª–µ–Ω–∏—è –≤—ã –∑–Ω–∞–µ—Ç–µ (Git Flow, GitHub Flow)?
**–û—Ç–≤–µ—Ç:**
*   **Git Flow:** –°–ª–æ–∂–Ω–∞—è —Å—Ç—Ä–∞—Ç–µ–≥–∏—è —Å –¥–æ–ª–≥–æ–∂–∏–≤—É—â–∏–º–∏ –≤–µ—Ç–∫–∞–º–∏ (`develop`, `main`) –∏ –º–Ω–æ–∂–µ—Å—Ç–≤–æ–º –≤—Å–ø–æ–º–æ–≥–∞—Ç–µ–ª—å–Ω—ã—Ö (`feature/*`, `release/*`, `hotfix/*`). –ü–æ–¥—Ö–æ–¥–∏—Ç –¥–ª—è –ø—Ä–æ–µ–∫—Ç–æ–≤ —Å —á–µ—Ç–∫–∏–º —Ü–∏–∫–ª–æ–º —Ä–µ–ª–∏–∑–æ–≤.
*   **GitHub Flow:** –ü—Ä–æ—Å—Ç–∞—è —Å—Ç—Ä–∞—Ç–µ–≥–∏—è. –ï—Å—Ç—å –æ–¥–Ω–∞ –≥–ª–∞–≤–Ω–∞—è –≤–µ—Ç–∫–∞ (`main`) –∏ –≤–µ—Ç–∫–∏ —Ñ–∏—á. –ü–æ—Å–ª–µ –∑–∞–≤–µ—Ä—à–µ–Ω–∏—è —Ä–∞–±–æ—Ç—ã –Ω–∞–¥ —Ñ–∏—á–µ–π —Å–æ–∑–¥–∞–µ—Ç—Å—è Pull Request. –ü–æ—Å–ª–µ —Ä–µ–≤—å—é –∏ –ø—Ä–æ–≤–µ—Ä–æ–∫ –≤–µ—Ç–∫–∞ —Å–ª–∏–≤–∞–µ—Ç—Å—è –≤ `main` –∏ —Å—Ä–∞–∑—É –¥–µ–ø–ª–æ–∏—Ç—Å—è. –ü–æ–¥—Ö–æ–¥–∏—Ç –¥–ª—è –≤–µ–±-–ø—Ä–æ–µ–∫—Ç–æ–≤ –∏ –Ω–µ–ø—Ä–µ—Ä—ã–≤–Ω–æ–π –¥–æ—Å—Ç–∞–≤–∫–∏ (CI/CD).
*   **GitLab Flow:** –í–∞—Ä–∏–∞—Ü–∏—è –Ω–∞ —Ç–µ–º—É, –∫–æ—Ç–æ—Ä–∞—è –¥–æ–±–∞–≤–ª—è–µ—Ç —Å—Ä–µ–¥—ã (–æ–∫—Ä—É–∂–µ–Ω–∏—è) –∏–ª–∏ –≤–µ—Ç–∫–∏ –¥–ª—è —Å–∏–Ω—Ö—Ä–æ–Ω–∏–∑–∞—Ü–∏–∏ —Å issues.
